PyGame を用いて作成した、ボンバーマン風の2Dアクションゲームです。  
プレイヤーは爆弾を設置し、敵を倒しながらステージを進んでいきます。

---

## 🎮 ゲーム概要

- グリッド状のマップを移動
- 爆弾を設置して敵や壊せるブロックを破壊
- 敵を全滅させると次のステージへ進行

---

## 🕹 操作方法

矢印キー：プレイヤー移動
Space  : 爆弾を設置

---

## 💣 ゲーム仕様
-赤がプレイヤー、青は敵。
- プレイヤーは同時に1つまで爆弾を設置可能
- 爆弾は一定時間後に爆発し、十字方向に爆風が発生
- 壊せるブロック（ひび割れブロック）は爆風で破壊が可能
- 爆風に当たるとプレイヤー・敵は消滅
- 敵を全て倒すとステージクリア

---

## 🗺 ステージ構成

- **Stage 1**：基本マップ(敵一体)
- **Stage 2**：敵が2体
- ステージが進むと敵の数が増加

---

## 🛠 使用技術

- Python 3.9.6
- PyGame 2.6.1

## 工夫したポイント!!
ボンバーマンらしい爆弾・爆風処理の実装
爆弾にタイマーを持たせ、一定時間後に爆発し、上下左右に爆風が広がる仕組みを実装した。
また、爆弾は設置直後は通過可能だが、設置後は障害物として機能するようにし、原作に近い挙動を再現した。

壊せる壁とマップの状態変化
固定壁と破壊可能な壁を区別してマップに配置し、爆風が当たった場合のみ破壊可能な壁が消えるようにした。
これにより、プレイヤーの行動によってマップ構造が変化するゲーム性を実現している。

ステージ制によるゲーム進行
敵をすべて倒すと次のステージへ進む構成にし、ステージごとに敵の数を増やすことで難易度が段階的に上がるよう工夫した。

クラス分割による管理のしやすさ
Player、Enemy、Bomb、Flame、GameMap などをクラスごとに分割し、それぞれの役割を明確にした。
これにより、処理の見通しが良くなり、機能追加や修正がしやすい構成となっている。

クールダウン処理による操作性の調整
プレイヤーや敵の移動・爆弾設置にクールダウンを設け、連続操作による不自然な挙動を防いだ。
ゲームとしての操作感を意識した実装である。
